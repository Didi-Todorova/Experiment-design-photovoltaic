# Photovoltaic System Data Analysis and Forecasting

## Overview

This project implements a complete data analysis and forecasting pipeline for a photovoltaic system efficiency dataset.

The workflow covers the entire process from raw data preprocessing to statistical analysis, predictive modeling, time-series forecasting, and numerical optimization.

The project was developed in Python as part of a university course on Experimental Design.

---

## Features

- Data cleaning
- Missing value detection and interpolation
- Outlier detection
- Signal smoothing
- Descriptive statistical analysis
- Correlation analysis
- Linear and quadratic regression models
- Model evaluation (MAE, RMSE, Bias)
- AR(1) residual forecasting
- Projected Gradient optimization
